## Verified non-issues

- **SQL execution:** 71 `ExecuteSQLTool` calls, zero content-level failures.
- **Docs-search retrieval relevance:** sampled outputs matched queries, grounded in results with source links.
- **Adversarial/garbage input handling:** 9 gibberish messages + lorem ipsum all got graceful, on-brand redirects (no crash, no hijack). The one "your message may have been cut short" reply initially looked like a misfire but was **correct** — the user really sent a stray `h`.
- **Truncation:** no length-stop/max-token cutoffs observed on outputs.
- **User feedback polarity:** all 7 feedback scores positive; no thumbs-down in window.
- **Utility surfaces** (title suggestion, chart config): fast (2–4s), cheap, sane outputs matching instructions (modulo fences, finding 9).
- **Language mismatch, PII/secret leakage in sampled content:** none observed (sampled, not exhaustive).
- **Evaluator wiring:** `Valid Plan Schema` comments show it sees real plans — the 56% False rate is a genuine product signal, not a mis-wired judge.
