## Verified non-issues (checked, held up)

- **`out_of_scope` evaluator** — valid and working. Reads the user message (all it needs); comments correctly identify off-topic (World Cup, weather). 18/228 True are genuine. (The *agent's handling* of them is the issue in #5, not the judge.)
- **`user-intent` evaluator** — valid and working; comments coherent and match inputs across all 6 categories.
- **`is_cursing` evaluator** — valid (reads user message); 100% False = **legitimately quiet** (no profanity in inputs over the window), confirmed via comments. Suggest a known-positive sanity test since it has never fired (P3 hygiene).
- **Language / generation mismatch** — clean. Chinese, Turkish, Spanish, Portuguese inputs all received same-language replies.
- **Output truncation** — negligible (1/263 heuristic; output tokens p90 492, max 1783 — well under limits).
- **Cost explosions** — none. Max single trace $0.166; total $9.04 / 7 days.
- **Token runaway** — none. Gen input max 33k, output max 1.8k; no context blow-up.
- **Safety refusal** — a bomb-making request was correctly refused (trace db29993b).
- **Retrieval relevance (dominant query)** — `searchLangfuseDocs` returns appropriate overview docs for the dominant "What can I use Langfuse for?" question; 24 distinct docs seen. Not exhaustively verified across the long tail (see Not covered).
